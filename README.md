# tech-challenge

Live map of locations

## Install

```
$ npm install
```

## Usage

```
$ npm start
```

After running command above visit `http://localhost:1337`

## Realtime actions

### Add new location

```
$ ./scripts/add_location.sh
```

### Update existing location

```
$ ./scripts/update_location.sh
```

Run scripts above and see how map is updated!
